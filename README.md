# Monte Carlo Ray Tracing

Features:
1. Acceleration data structure - Uniform Grid, Fast voxel traversal algorithm
2. 5000 samples per pixel
3. Bounce depth - 5 
4. Effects: soft shadows, reflections, refractions, global illumination
5. Denoisation
